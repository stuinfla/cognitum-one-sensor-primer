# ruvector-kb.rvf — Manifest

RVF knowledge base over the **RuVector monorepo** — docs, ADRs, research, crate
manifests/READMEs, examples, npm packages, skills, **and (v2) the Rust crates' actual
implementation knowledge**. Built by mechanical enumeration — no sampling — so nested
sub-workspaces, research subtrees, and source-level doc comments are not missed.

Rust crates are the actual solution — v2 indexes both. The v1 store is preserved as
`ruvector-kb.v1.rvf` (+ `.v1` sidecars).
